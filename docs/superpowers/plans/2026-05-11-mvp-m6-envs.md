# Lancer MVP M6 — Environments and Variable Substitution Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use `superpowers:subagent-driven-development` (recommended) or `superpowers:executing-plans` to execute this plan task-by-task.

**Goal:** Make Lancer's URL, headers, params, body, and auth fields accept template strings (`{{varName}}`) and resolve them against the **selected environment** at send time. Environments are stored as plain `.bru` files in an `environments/` folder inside the workspace (Bruno-compatible). Secret values are kept out of disk by storing them in the OS keyring (Windows Credential Manager / macOS Keychain / Linux Secret Service).

**Architecture:**
- Environments live in `<workspace>/environments/*.bru`. Each file is one environment: `dev.bru`, `staging.bru`, `prod.bru`. The user can name them anything.
- File format (Bruno-compatible):
  ```
  vars {
    baseUrl: https://api.example.com
    apiVersion: v1
  }

  vars:secret [
    apiKey,
    oauthSecret
  ]
  ```
  The `vars:secret` block is a **list** of variable names whose values live in OS keyring, NOT in this file. The file commits to Git safely; the keyring stays per-machine.
- Variable substitution is a **single-pass, non-recursive** rewrite of every string field in a `Request` (URL, header values, param values, body content, auth field values, vars values). `{{name}}` lookups go through `vars` first, then through `secrets` (keyring). If neither matches, the template is left as-is and an informational warning may be surfaced in the future (not yet — silent leave-as-is for now).
- Substitution runs in Rust **before** `apply_auth`, so auth field values can themselves contain templates.
- The Tauri command `send_request` grows a third optional argument: `env_name: Option<String>`. The frontend stores the active env name in `localStorage` keyed by the workspace root path. When the user changes env, the next send picks it up.
- A `<EnvSwitcher>` shadcn `<Select>` lives in the top-right of the editor pane. A "Manage environments…" item opens a side `<Sheet>` with a per-env editor.

**Tech Stack additions:**
- `keyring` (Rust crate, MIT/Apache) — cross-platform credential store wrapper. Pin to `^3`.
- No new JS deps.

```rust
use crate::env::schema::Environment;

#[derive(Debug, thiserror::Error)]
pub enum EnvBruError {
    #[error("malformed line: {0}")]
    MalformedLine(String),
    #[error("unterminated block: {0}")]
    UnterminatedBlock(String),
}

/// Parse an env `.bru` file. `name` is the environment name (typically the
/// filename without extension).
pub fn parse(name: &str, input: &str) -> Result<Environment, EnvBruError> {
    let mut vars = Vec::<(String, String)>::new();
    let mut secret_names = Vec::<String>::new();
    let mut chars = input.char_indices().peekable();

    while let Some(&(_, c)) = chars.peek() {
        if c.is_whitespace() {
            chars.next();
            continue;
        }
        // Read block header up to `{` or `[`
        let header_start = chars.next().unwrap().0;
        let mut header_end = header_start + c.len_utf8();
        while let Some(&(_, ch)) = chars.peek() {
            if ch == '{' || ch == '[' {
                break;
            }
            chars.next();
            header_end += ch.len_utf8();
        }
        let header = input[header_start..header_end].trim().to_string();
        let Some(&(_, opener)) = chars.peek() else {
            return Err(EnvBruError::UnterminatedBlock(header));
        };
        let closer = match opener {
            '{' => '}',
            '[' => ']',
            _ => return Err(EnvBruError::UnterminatedBlock(header)),
        };
        chars.next(); // consume opener

        let body_start = chars.peek().map(|(i, _)| *i).unwrap_or(input.len());
        let mut body_end = body_start;
        loop {
            match chars.next() {
                Some((idx, ch)) if ch == closer => {
                    body_end = idx;
                    break;
                }
                Some((idx, _)) => {
                    body_end = idx + 1;
                }
                None => return Err(EnvBruError::UnterminatedBlock(header)),
            }
        }
        let body = input[body_start..body_end].trim();

        match header.as_str() {
            "vars" => {
                for raw in body.lines() {
                    let line = raw.trim();
                    if line.is_empty() {
                        continue;
                    }
                    let (k, v) = line
                        .split_once(':')
                        .ok_or_else(|| EnvBruError::MalformedLine(line.to_string()))?;
                    vars.push((k.trim().to_string(), v.trim().to_string()));
                }
            }
            "vars:secret" => {
                for raw in body.split(|c: char| c == ',' || c == '\n') {
                    let name = raw.trim();
                    if !name.is_empty() {
                        secret_names.push(name.to_string());
                    }
                }
            }
            _ => {
                // Unknown block — ignore for forward compatibility
            }
        }
    }

    Ok(Environment {
        name: name.to_string(),
        vars,
        secret_names,
    })
}

pub fn serialize(env: &Environment) -> String {
    let mut out = String::new();
    if !env.vars.is_empty() {
        out.push_str("vars {\n");
        for (k, v) in &env.vars {
            out.push_str(&format!("  {k}: {v}\n"));
        }
        out.push_str("}\n\n");
    }
    if !env.secret_names.is_empty() {
        out.push_str("vars:secret [\n");
        for (i, name) in env.secret_names.iter().enumerate() {
            if i + 1 == env.secret_names.len() {
                out.push_str(&format!("  {name}\n"));
            } else {
                out.push_str(&format!("  {name},\n"));
            }
        }
        out.push_str("]\n");
    }
    out
}
```

```rust
use crate::env::bru;
use crate::env::schema::Environment;
use std::fs;
use std::path::Path;

#[derive(Debug, thiserror::Error)]
pub enum EnvIoError {
    #[error("io: {0}")]
    Io(#[from] std::io::Error),
    #[error("parse: {0}")]
    Parse(#[from] crate::env::bru::EnvBruError),
}

impl serde::Serialize for EnvIoError {
    fn serialize<S: serde::Serializer>(&self, s: S) -> Result<S::Ok, S::Error> {
        s.serialize_str(&self.to_string())
    }
}

pub fn list_envs(workspace_root: &Path) -> Result<Vec<String>, EnvIoError> {
    let envs_dir = workspace_root.join("environments");
    let mut out = Vec::new();
    let Ok(entries) = fs::read_dir(&envs_dir) else {
        return Ok(out);
    };
    for entry in entries.filter_map(Result::ok) {
        let path = entry.path();
        if path.extension().and_then(|e| e.to_str()) != Some("bru") {
            continue;
        }
        if let Some(stem) = path.file_stem().and_then(|s| s.to_str()) {
            out.push(stem.to_string());
        }
    }
    out.sort();
    Ok(out)
}

pub fn read_env(workspace_root: &Path, name: &str) -> Result<Environment, EnvIoError> {
    let path = workspace_root.join("environments").join(format!("{name}.bru"));
    let text = fs::read_to_string(&path)?;
    Ok(bru::parse(name, &text)?)
}

pub fn write_env(workspace_root: &Path, env: &Environment) -> Result<(), EnvIoError> {
    let envs_dir = workspace_root.join("environments");
    fs::create_dir_all(&envs_dir)?;
    let path = envs_dir.join(format!("{}.bru", env.name));
    fs::write(path, bru::serialize(env))?;
    Ok(())
}

pub fn delete_env(workspace_root: &Path, name: &str) -> Result<(), EnvIoError> {
    let path = workspace_root.join("environments").join(format!("{name}.bru"));
    if path.exists() {
        fs::remove_file(path)?;
    }
    Ok(())
}
```

```rust
use std::collections::HashMap;

#[derive(Debug, Default, Clone)]
pub struct Ctx {
    pub vars: HashMap<String, String>,
    pub secrets: HashMap<String, String>,
}

impl Ctx {
    fn lookup(&self, name: &str) -> Option<&str> {
        self.secrets
            .get(name)
            .or_else(|| self.vars.get(name))
            .map(String::as_str)
    }
}

/// Single-pass `{{name}}` substitution. Unknown names are left as the literal
/// `{{name}}` text (no error).
pub fn substitute(input: &str, ctx: &Ctx) -> String {
    let mut out = String::with_capacity(input.len());
    let bytes = input.as_bytes();
    let mut i = 0usize;
    while i < bytes.len() {
        // Look for `{{`
        if i + 1 < bytes.len() && bytes[i] == b'{' && bytes[i + 1] == b'{' {
            // Find the matching `}}`
            if let Some(close_rel) = input[i + 2..].find("}}") {
                let name = input[i + 2..i + 2 + close_rel].trim();
                if let Some(value) = ctx.lookup(name) {
                    out.push_str(value);
                } else {
                    // Leave literal `{{name}}` (preserve exact original spans
                    // so re-substitution later works).
                    out.push_str(&input[i..i + 2 + close_rel + 2]);
                }
                i += 2 + close_rel + 2;
                continue;
            }
        }
        // Default: copy one byte/char.
        let ch = input[i..].chars().next().unwrap();
        out.push(ch);
        i += ch.len_utf8();
    }
    out
}
```

### Task 6.2.2 — Apply substitution to a `collection::schema::Request` + `Auth`

The substitute function works on a single `&str`. We need a wrapper that walks every templatable field in the on-disk `collection::schema::Request` form — **not** in `http::types::HttpRequest`. Substitution happens on the raw, pre-materialized request so that template injection is caught at JSON parse time rather than silently succeeding.

**Security note — JSON body substitution:** The `collection::schema::RequestBody::Json { value: String }` variant stores the user's raw JSON string as authored on disk. Substitution runs on this raw string literal **before** any JSON parsing. This means that if `{{nonce}}` resolves to `","admin":true,"`, the resulting string is not valid JSON and the subsequent parse (done in the `materialize` step, Task 6.2.3) will fail with a `ConvertError::InvalidJson` — the request is rejected rather than silently mutated. This prevents template-injection attacks against the JSON body. **Do not** substitute on a pre-parsed `serde_json::Value` then re-serialize: that approach allows injected strings to escape JSON quoting at the leaf level.

- [ ] **Step 1: Helper in `env/substitute.rs`**

```rust
use crate::collection::schema::{Auth, Request, RequestBody};

/// Walks every templatable string in a `collection::schema::Request` (the
/// on-disk, pre-materialized form) and substitutes `{{name}}` in-place.
/// JSON body substitution operates on the raw string literal authored by the
/// user — see the security note in the plan for why this matters.
pub fn substitute_request(req: &mut Request, ctx: &Ctx) {
    req.url = substitute(&req.url, ctx);
    for kv in req.headers.iter_mut() {
        kv.key = substitute(&kv.key, ctx);
        kv.value = substitute(&kv.value, ctx);
    }
    for kv in req.params.iter_mut() {
        kv.key = substitute(&kv.key, ctx);
        kv.value = substitute(&kv.value, ctx);
    }
    if let Some(body) = req.body.as_mut() {
        use crate::collection::schema::RequestBody::*;
        match body {
            Json { value } => *value = substitute(value, ctx),
            Text { value, .. } => *value = substitute(value, ctx),
            FormUrlencoded { fields } | MultipartForm { fields } => {
                for kv in fields.iter_mut() {
                    kv.key = substitute(&kv.key, ctx);
                    kv.value = substitute(&kv.value, ctx);
                }
            }
            GraphQl { query, variables } => {
                *query = substitute(query, ctx);
                *variables = substitute(variables, ctx);
            }
        }
    }
}

pub fn substitute_auth(auth: &mut Auth, ctx: &Ctx) {
    match auth {
        Auth::None => {}
        Auth::Bearer { token } => *token = substitute(token, ctx),
        Auth::Basic { username, password } => {
            *username = substitute(username, ctx);
            *password = substitute(password, ctx);
        }
```

### Task 6.2.3 — Materialize `collection::schema::Request` → `http::types::HttpRequest`

After substitution is applied to the on-disk `collection::schema::Request`, a separate `materialize` step converts it to the wire-ready `http::types::HttpRequest`. This separation keeps substitution pure (string-in, string-out, no side effects) and ensures JSON is re-parsed after substitution — catching injection attempts with a parse error rather than silently accepting them.

- [ ] **Step 1: Implement `materialize` in `env/substitute.rs` (or a new `env/materialize.rs`)**

```rust
use crate::collection::schema::{Request as SchemaRequest, RequestBody as SchemaBody};
use crate::http::types::{HttpRequest, RequestBody as HttpBody};

#[derive(Debug, thiserror::Error)]
pub enum ConvertError {
    #[error("invalid JSON in body after substitution: {0}")]
    InvalidJson(#[from] serde_json::Error),
    #[error("unsupported method: {0}")]
    BadMethod(String),
}

/// Convert a (already-substituted) `collection::schema::Request` into the
/// wire-ready `http::types::HttpRequest`. JSON body is re-parsed here; if
/// substitution injected invalid JSON the parse error surfaces as
/// `ConvertError::InvalidJson`.
pub fn materialize(req: &SchemaRequest, _ctx: &Ctx) -> Result<HttpRequest, ConvertError> {
    let mut http_req = HttpRequest::new(&req.url);
    http_req.method = parse_method(&req.method).map_err(ConvertError::BadMethod)?;
    for kv in &req.headers {
        if kv.enabled {
            http_req.headers.push((kv.key.clone(), kv.value.clone()));
        }
    }
    for kv in &req.params {
        if kv.enabled {
            http_req.query.push((kv.key.clone(), kv.value.clone()));
        }
    }
    http_req.body = match &req.body {
        None => None,
        Some(SchemaBody::Json { value }) => {
            // Parse the substituted string — injection caught here if invalid JSON.
            let parsed: serde_json::Value = serde_json::from_str(value)?;
            Some(HttpBody::Json { value: parsed })
        }
        Some(SchemaBody::Text { value, content_type }) => Some(HttpBody::Text {
            value: value.clone(),
            content_type: content_type.clone(),
        }),
        Some(SchemaBody::FormUrlencoded { fields }) => Some(HttpBody::Form {
            fields: fields.iter().filter(|kv| kv.enabled)
                .map(|kv| (kv.key.clone(), kv.value.clone()))
                .collect(),
        }),
        Some(SchemaBody::MultipartForm { fields }) => Some(HttpBody::Multipart {
            fields: fields.iter().filter(|kv| kv.enabled)
                .map(|kv| (kv.key.clone(), kv.value.clone()))
                .collect(),
        }),
        Some(SchemaBody::GraphQl { query, variables }) => Some(HttpBody::GraphQl {
            query: query.clone(),
            variables: variables.clone(),
        }),
    };
    Ok(http_req)
}
```

The Tauri command now receives a `collection::schema::Request` (the on-disk form), applies substitution to it, then materializes to `http::types::HttpRequest` before sending. This is the correct pipeline order:

1. Load `collection::schema::Request` from `.bru`
2. `substitute_request(&mut req, &ctx)` — all `{{}}` resolved on raw strings
3. `substitute_auth(auth, &ctx)` — auth fields resolved
4. `materialize(&req, &ctx)` — JSON re-parsed, wire request built
5. `apply_auth` + `client::send`

```rust
use std::path::PathBuf;
use crate::collection::schema::{Auth, Request as SchemaRequest};
use crate::env::{io as env_io, secrets as env_secrets, substitute};
use crate::http::{auth, client, types::HttpResponse};
use crate::state::AppState;

#[tauri::command]
pub async fn send_request(
    mut req: SchemaRequest,
    mut auth: Option<Auth>,
    workspace_root: Option<PathBuf>,
    env_name: Option<String>,
    state: tauri::State<'_, AppState>,
) -> Result<HttpResponse, String> {
    // 1. Build substitution context from env file + keyring secrets, if any.
    if let (Some(root), Some(name)) = (workspace_root.as_ref(), env_name.as_ref()) {
        let env = env_io::read_env(root, name).map_err(|e| e.to_string())?;
        let mut ctx = substitute::Ctx::default();
        for (k, v) in &env.vars {
            ctx.vars.insert(k.clone(), v.clone());
        }
        for secret_name in &env.secret_names {
            match env_secrets::get(root, name, secret_name) {
                Ok(Some(v)) => { ctx.secrets.insert(secret_name.clone(), v); }
                Ok(None) => {}
                Err(e) => return Err(format!("keyring unavailable for {secret_name}: {e}")),
            }
        }
        substitute::substitute_request(&mut req, &ctx);
        if let Some(a) = auth.as_mut() {
            substitute::substitute_auth(a, &ctx);
        }
    }

    // 2. Materialize to wire request (JSON re-parsed here — injection caught).
    let http_req = substitute::materialize(&req, &substitute::Ctx::default())
        .map_err(|e| e.to_string())?;

    // 3. Apply auth, then send.
    let http_req = match auth {
        Some(a) => auth::apply_auth(http_req, &a, state.inner())
            .await
            .map_err(|e| e.to_string())?,
        None => http_req,
    };
    client::send(&state.http_client, http_req).await.map_err(|e| e.to_string())
}
```

```rust
use keyring::Entry;
use std::path::Path;

#[derive(Debug, thiserror::Error)]
pub enum SecretError {
    #[error("keyring error: {0}")]
    Keyring(#[from] keyring::Error),
    #[error("invalid path: {0}")]
    InvalidPath(String),
    #[error("keyring unavailable: {0}")]
    Unavailable(String),
}

impl serde::Serialize for SecretError {
    fn serialize<S: serde::Serializer>(&self, s: S) -> Result<S::Ok, S::Error> {
        s.serialize_str(&self.to_string())
    }
}

const SERVICE_PREFIX: &str = "dev.lancer.app";

/// Build a keyring entry. The service name uses a 16-hex-char SHA-256 prefix
/// of the workspace path to stay well under Windows Credential Manager's
/// 256-character target-name limit (a 200-char path would otherwise exhaust
/// the budget when combined with the env name and prefix).
fn entry_for(
    workspace_root: &Path,
    env_name: &str,
    var_name: &str,
) -> Result<Entry, SecretError> {
    let root_str = workspace_root
        .to_str()
        .ok_or_else(|| SecretError::InvalidPath(format!("{workspace_root:?}")))?;
    // Hash the workspace root to fit Windows Credential Manager's 256-char target-name limit.
    use sha2::{Digest, Sha256};
    let mut h = Sha256::new();
    h.update(root_str.as_bytes());
    let hash = format!("{:x}", h.finalize());
    let short_hash = &hash[..16]; // first 16 hex chars — 64 bits of entropy
    let service = format!("{SERVICE_PREFIX}:{short_hash}:{env_name}");
    Ok(Entry::new(&service, var_name)?)
}

pub fn get(
    workspace_root: &Path,
    env_name: &str,
    var_name: &str,
) -> Result<Option<String>, SecretError> {
    let entry = entry_for(workspace_root, env_name, var_name)?;
    match entry.get_password() {
        Ok(p) => Ok(Some(p)),
        Err(keyring::Error::NoEntry) => Ok(None),
        Err(keyring::Error::PlatformFailure(e)) => Err(SecretError::Unavailable(e.to_string())),
        Err(keyring::Error::NoStorageAccess(e)) => Err(SecretError::Unavailable(e.to_string())),
        Err(e) => Err(SecretError::Keyring(e)),
    }
}

// When the keyring is locked or unavailable, `send_request` MUST fail the
// request rather than silently using an unsubstituted `{{name}}` template —
// this prevents information leakage to the upstream API. The `Unavailable`
// variant propagates through `get()` and the caller in `commands/http.rs`
// converts it to an `Err` that stops the send.

pub fn set(
    workspace_root: &Path,
    env_name: &str,
    var_name: &str,
    value: &str,
) -> Result<(), SecretError> {
    let entry = entry_for(workspace_root, env_name, var_name)?;
    entry.set_password(value)?;
    Ok(())
}

pub fn delete(
    workspace_root: &Path,
    env_name: &str,
    var_name: &str,
) -> Result<(), SecretError> {
    let entry = entry_for(workspace_root, env_name, var_name)?;
    match entry.delete_credential() {
        Ok(()) | Err(keyring::Error::NoEntry) => Ok(()),
        Err(e) => Err(SecretError::Keyring(e)),
    }
}
```

```ts
// src/stores/env-store.ts
import { create } from "zustand";
import { listEnvs } from "@/lib/tauri";

interface EnvState {
  available: string[];
  activeEnv: string | null;
  loading: boolean;
  setActiveEnv: (name: string | null, workspaceRoot: string | null) => void;
  refresh: (workspaceRoot: string | null) => Promise<void>;
}

const STORAGE_KEY = (root: string) => `lancer.activeEnv:${root}`;

export const useEnv = create<EnvState>((set) => ({
  available: [],
  activeEnv: null,
  loading: false,
  setActiveEnv: (name, workspaceRoot) => {
    if (workspaceRoot) {
      if (name) localStorage.setItem(STORAGE_KEY(workspaceRoot), name);
      else localStorage.removeItem(STORAGE_KEY(workspaceRoot));
    }
    set({ activeEnv: name });
  },
  refresh: async (workspaceRoot) => {
    if (!workspaceRoot) {
      set({ available: [], activeEnv: null });
      return;
    }
    set({ loading: true });
    try {
      const available = await listEnvs(workspaceRoot);
      const stored = localStorage.getItem(STORAGE_KEY(workspaceRoot));
      const activeEnv = stored && available.includes(stored) ? stored : null;
      set({ available, activeEnv, loading: false });
    } catch {
      set({ available: [], loading: false });
    }
  },
}));
```
