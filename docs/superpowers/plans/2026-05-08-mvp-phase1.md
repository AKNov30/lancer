# Lancer MVP (Phase 1) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use `superpowers:subagent-driven-development` (recommended) or `superpowers:executing-plans` to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship a free, local-first desktop API client (working name "Lancer") that supports REST, collections persisted as `.bru` files, OpenAPI mock, basic post-response tests, and Postman v2.1 import — running entirely on the user's machine with zero infrastructure required from the maintainer.

**Architecture:** Tauri 2 desktop shell. **Rust core** (in `src-tauri/`) owns HTTP, file I/O, `.bru` parsing/writing, sandboxed JS test runner (`rquickjs`), and the optional embedded mock server (`axum`). **React 19 frontend** (in `src/`) renders UI via Tailwind v4 + shadcn/ui (new-york). Communication strictly via Tauri commands (IPC). Collections live in plain files in a user-chosen folder; sync = the user's Git.

**Tech Stack:** Tauri 2.x · Rust 1.85 · React 19 · TypeScript 5.6 · Vite 7 · Tailwind v4 · shadcn/ui (new-york, radix base) · pnpm · reqwest · axum · rquickjs · serde · nom · rusqlite · Vitest · cargo nextest · WebDriverIO · Biome · Lefthook.

Create `src-tauri/src/http/client.rs`:

```rust
use crate::http::types::{HttpRequest, HttpResponse, Method, RequestBody};
use std::time::Instant;
use thiserror::Error;

#[derive(Debug, Error)]
pub enum HttpError {
    #[error("invalid url: {0}")]
    InvalidUrl(String),
    #[error("network error: {0}")]
    Network(#[from] reqwest::Error),
}

impl serde::Serialize for HttpError {
    fn serialize<S: serde::Serializer>(&self, s: S) -> Result<S::Ok, S::Error> {
        s.serialize_str(&self.to_string())
    }
}

pub async fn send(req: HttpRequest) -> Result<HttpResponse, HttpError> {
    let client = reqwest::Client::builder()
        .user_agent(concat!("Lancer/", env!("CARGO_PKG_VERSION")))
        .cookie_store(true)
        .build()?;

    let method = match req.method {
        Method::Get => reqwest::Method::GET,
        Method::Post => reqwest::Method::POST,
        Method::Put => reqwest::Method::PUT,
        Method::Patch => reqwest::Method::PATCH,
        Method::Delete => reqwest::Method::DELETE,
        Method::Head => reqwest::Method::HEAD,
        Method::Options => reqwest::Method::OPTIONS,
    };

    let mut builder = client.request(method, &req.url);
    for (k, v) in &req.headers {
        builder = builder.header(k, v);
    }
    if !req.query.is_empty() {
        builder = builder.query(&req.query);
    }
    builder = match req.body {
        Some(RequestBody::Json { value }) => builder.json(&value),
        Some(RequestBody::Text { value, content_type }) => {
            builder.header("content-type", content_type).body(value)
        }
        Some(RequestBody::Form { fields }) => builder.form(&fields),
        Some(RequestBody::None) | None => builder,
    };

    let start = Instant::now();
    let resp = builder.send().await?;
    let status = resp.status().as_u16();
    let status_text = resp
        .status()
        .canonical_reason()
        .unwrap_or("")
        .to_string();
    let headers: Vec<(String, String)> = resp
        .headers()
        .iter()
        .map(|(k, v)| (k.to_string(), v.to_str().unwrap_or("").to_string()))
        .collect();
    let body = resp.bytes().await?.to_vec();
    let elapsed_ms = start.elapsed().as_millis();
    let size_bytes = body.len();
    let body_text = std::str::from_utf8(&body).ok().map(String::from);

    Ok(HttpResponse {
        status,
        status_text,
        headers,
        body,
        body_text,
        elapsed_ms,
        size_bytes,
    })
}
```

Create `src-tauri/src/collection/bru.rs`. Use a hand-written line-based parser (simpler than `nom` for this format and easier to evolve):

```rust
use crate::collection::schema::{Auth, KvEnabled, Request, RequestBody};
use crate::http::types::Method;
use std::collections::HashMap;

#[derive(Debug, thiserror::Error)]
pub enum BruError {
    #[error("missing block: {0}")]
    MissingBlock(&'static str),
    #[error("invalid block: {0}")]
    InvalidBlock(String),
    #[error("malformed line: {0}")]
    MalformedLine(String),
}

#[derive(Debug, Default)]
struct Blocks {
    map: HashMap<String, String>,
}

fn split_blocks(input: &str) -> Result<Blocks, BruError> {
    let mut blocks = Blocks::default();
    let mut chars = input.char_indices().peekable();
    while let Some((i, c)) = chars.next() {
        if c.is_whitespace() { continue; }
        let header_start = i;
        // read header until '{'
        let mut header_end = header_start;
        while let Some(&(_, ch)) = chars.peek() {
            if ch == '{' { break; }
            chars.next();
            header_end += ch.len_utf8();
        }
        let header = input[header_start..header_end].trim().to_string();
        // consume '{'
        if !matches!(chars.next(), Some((_, '{'))) {
            return Err(BruError::InvalidBlock(header));
        }
        // read body until matching '}', with depth count for safety
        let body_start_byte = chars.peek().map(|(idx, _)| *idx).unwrap_or(input.len());
        let mut depth = 1usize;
        let mut body_end_byte = body_start_byte;
        while let Some((idx, ch)) = chars.next() {
            match ch {
                '{' => depth += 1,
                '}' => {
                    depth -= 1;
                    if depth == 0 {
                        body_end_byte = idx;
                        break;
                    }
                }
                _ => {}
            }
        }
        let body = input[body_start_byte..body_end_byte].trim_matches('\n').to_string();
        blocks.map.insert(header, body);
    }
    Ok(blocks)
}

fn parse_kv_block(text: &str) -> Result<HashMap<String, String>, BruError> {
    let mut out = HashMap::new();
    for raw in text.lines() {
        let line = raw.trim();
        if line.is_empty() { continue; }
        let (k, v) = line
            .split_once(':')
            .ok_or_else(|| BruError::MalformedLine(line.to_string()))?;
        out.insert(k.trim().to_string(), v.trim().to_string());
    }
    Ok(out)
}

fn parse_kv_list(text: &str) -> Result<Vec<KvEnabled>, BruError> {
    let mut out = Vec::new();
    for raw in text.lines() {
        let line = raw.trim();
        if line.is_empty() { continue; }
        let enabled = !line.starts_with('~');
        let line = line.trim_start_matches('~').trim();
        let (k, v) = line
            .split_once(':')
            .ok_or_else(|| BruError::MalformedLine(line.to_string()))?;
        out.push(KvEnabled {
            key: k.trim().to_string(),
            value: v.trim().to_string(),
            enabled,
        });
    }
    Ok(out)
}

pub fn parse(input: &str) -> Result<Request, BruError> {
    let blocks = split_blocks(input)?;

    let meta = blocks.map.get("meta").ok_or(BruError::MissingBlock("meta"))?;
    let meta_kv = parse_kv_block(meta)?;
    let name = meta_kv.get("name").cloned().unwrap_or_default();
    let seq = meta_kv.get("seq").and_then(|s| s.parse().ok());

    // pick the http method block
    let methods = ["get", "post", "put", "patch", "delete", "head", "options"];
    let (method_str, mblock) = methods
        .iter()
        .find_map(|m| blocks.map.get(*m).map(|b| (*m, b.clone())))
        .ok_or(BruError::MissingBlock("<method>"))?;
    let m_kv = parse_kv_block(&mblock)?;
    let url = m_kv.get("url").cloned().unwrap_or_default();
    let body_marker = m_kv.get("body").cloned();
    let auth_marker = m_kv.get("auth").cloned();
    let method = match method_str {
        "get" => Method::Get, "post" => Method::Post, "put" => Method::Put,
        "patch" => Method::Patch, "delete" => Method::Delete,
        "head" => Method::Head, "options" => Method::Options,
        _ => unreachable!(),
    };

    let headers = blocks
        .map
        .get("headers")
        .map(|s| parse_kv_list(s))
        .transpose()?
        .unwrap_or_default();
    let params = blocks
        .map
        .get("params:query")
        .map(|s| parse_kv_list(s))
        .transpose()?
        .unwrap_or_default();
    let vars = blocks
        .map
        .get("vars:pre-request")
        .map(|s| parse_kv_list(s))
        .transpose()?
        .unwrap_or_default();

    let auth = match auth_marker.as_deref() {
        Some("bearer") => {
            let kv = parse_kv_block(blocks.map.get("auth:bearer").ok_or(BruError::MissingBlock("auth:bearer"))?)?;
            Some(Auth::Bearer { token: kv.get("token").cloned().unwrap_or_default() })
        }
        Some("basic") => {
            let kv = parse_kv_block(blocks.map.get("auth:basic").ok_or(BruError::MissingBlock("auth:basic"))?)?;
            Some(Auth::Basic {
                username: kv.get("username").cloned().unwrap_or_default(),
                password: kv.get("password").cloned().unwrap_or_default(),
            })
        }
        Some("none") | None => Some(Auth::None),
        Some(other) => return Err(BruError::InvalidBlock(format!("unknown auth: {other}"))),
    };

    let body = match body_marker.as_deref() {
        Some("json") => blocks.map.get("body:json").map(|s| RequestBody::Json { value: s.trim().to_string() }),
        Some("text") => blocks.map.get("body:text").map(|s| RequestBody::Text {
            value: s.trim().to_string(),
            content_type: "text/plain".to_string(),
        }),
        _ => None,
    };

    Ok(Request {
        name, seq, method, url, headers, params, body, auth, vars,
    })
}
```

```rust
pub fn serialize(req: &Request) -> String {
    let mut out = String::new();

    out.push_str("meta {\n");
    out.push_str(&format!("  name: {}\n", req.name));
    out.push_str("  type: http\n");
    if let Some(seq) = req.seq {
        out.push_str(&format!("  seq: {}\n", seq));
    }
    out.push_str("}\n\n");

    let method_str = match req.method {
        Method::Get => "get", Method::Post => "post", Method::Put => "put",
        Method::Patch => "patch", Method::Delete => "delete",
        Method::Head => "head", Method::Options => "options",
    };
    out.push_str(&format!("{method_str} {{\n"));
    out.push_str(&format!("  url: {}\n", req.url));
    let body_marker = match &req.body {
        Some(RequestBody::Json { .. }) => Some("json"),
        Some(RequestBody::Text { .. }) => Some("text"),
        Some(RequestBody::FormUrlencoded { .. }) => Some("form-urlencoded"),
        Some(RequestBody::MultipartForm { .. }) => Some("multipart-form"),
        Some(RequestBody::GraphQl { .. }) => Some("graphql"),
        None => None,
    };
    if let Some(b) = body_marker { out.push_str(&format!("  body: {}\n", b)); }
    let auth_marker = match &req.auth {
        Some(Auth::None) | None => "none",
        Some(Auth::Bearer { .. }) => "bearer",
        Some(Auth::Basic { .. }) => "basic",
        Some(Auth::ApiKey { .. }) => "apikey",
    };
    out.push_str(&format!("  auth: {}\n", auth_marker));
    out.push_str("}\n\n");

    if !req.headers.is_empty() {
        out.push_str("headers {\n");
        for h in &req.headers {
            out.push_str(&format!("  {}{}: {}\n", if h.enabled { "" } else { "~" }, h.key, h.value));
        }
        out.push_str("}\n\n");
    }
    if !req.params.is_empty() {
        out.push_str("params:query {\n");
        for p in &req.params {
            out.push_str(&format!("  {}{}: {}\n", if p.enabled { "" } else { "~" }, p.key, p.value));
        }
        out.push_str("}\n\n");
    }
    match &req.auth {
        Some(Auth::Bearer { token }) => {
            out.push_str(&format!("auth:bearer {{\n  token: {}\n}}\n\n", token));
        }
        Some(Auth::Basic { username, password }) => {
            out.push_str(&format!(
                "auth:basic {{\n  username: {}\n  password: {}\n}}\n\n",
                username, password
            ));
        }
        _ => {}
    }
    match &req.body {
        Some(RequestBody::Json { value }) => {
            out.push_str(&format!("body:json {{\n{}\n}}\n\n", value));
        }
        Some(RequestBody::Text { value, .. }) => {
            out.push_str(&format!("body:text {{\n{}\n}}\n\n", value));
        }
        _ => {}
    }
    if !req.vars.is_empty() {
        out.push_str("vars:pre-request {\n");
        for v in &req.vars {
            out.push_str(&format!("  {}{}: {}\n", if v.enabled { "" } else { "~" }, v.key, v.value));
        }
        out.push_str("}\n");
    }
    out
}
```

```rust
use crate::collection::bru;
use crate::collection::schema::Request;
use serde::Serialize;
use std::fs;
use std::path::{Path, PathBuf};
use walkdir::WalkDir;

#[derive(Debug, Clone, Serialize)]
#[serde(rename_all = "camelCase")]
pub struct WorkspaceItem {
    pub path: PathBuf,
    pub rel_path: String,
    pub name: String,
    pub method: String,
    pub seq: Option<u32>,
}

pub fn list_workspace(root: &Path) -> Result<Vec<WorkspaceItem>, std::io::Error> {
    let mut out = Vec::new();
    for entry in WalkDir::new(root).into_iter().filter_map(|e| e.ok()) {
        let path = entry.path();
        if path.is_file() && path.extension().and_then(|e| e.to_str()) == Some("bru") {
            let text = fs::read_to_string(path)?;
            if let Ok(req) = bru::parse(&text) {
                let rel = path.strip_prefix(root).unwrap_or(path).to_string_lossy().into_owned();
                out.push(WorkspaceItem {
                    path: path.to_path_buf(),
                    rel_path: rel,
                    name: req.name,
                    method: format!("{:?}", req.method).to_uppercase(),
                    seq: req.seq,
                });
            }
        }
    }
    out.sort_by(|a, b| a.rel_path.cmp(&b.rel_path));
    Ok(out)
}

pub fn read_request(path: &Path) -> Result<Request, String> {
    let text = fs::read_to_string(path).map_err(|e| e.to_string())?;
    bru::parse(&text).map_err(|e| e.to_string())
}

pub fn write_request(path: &Path, req: &Request) -> Result<(), String> {
    let text = bru::serialize(req);
    fs::write(path, text).map_err(|e| e.to_string())
}
```

```ts
import { create } from "zustand";
import { open } from "@tauri-apps/plugin-dialog";
import { listWorkspace, type WorkspaceItem } from "@/lib/tauri";

interface WorkspaceState {
  rootPath: string | null;
  items: WorkspaceItem[];
  loading: boolean;
  error: string | null;
  openFolder: () => Promise<void>;
  refresh: () => Promise<void>;
}

const STORAGE_KEY = "lancer.lastFolder";

export const useWorkspace = create<WorkspaceState>((set, get) => ({
  rootPath: localStorage.getItem(STORAGE_KEY),
  items: [],
  loading: false,
  error: null,
  async openFolder() {
    const picked = await open({ directory: true, multiple: false });
    if (typeof picked !== "string") return;
    localStorage.setItem(STORAGE_KEY, picked);
    set({ rootPath: picked, error: null });
    await get().refresh();
  },
  async refresh() {
    const { rootPath } = get();
    if (!rootPath) return;
    set({ loading: true, error: null });
    try {
      const items = await listWorkspace(rootPath);
      set({ items, loading: false });
    } catch (e) {
      set({ error: String(e), loading: false });
    }
  },
}));
```
